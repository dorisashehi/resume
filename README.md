# [CV Maker](https://resumemaker24.netlify.app/)

CV builder website. \
Concepts included: React.js, Vite, Netlify.

# Features

- Add/Edit education
- Add/Edit profile
- Add/Edit profile summary
- Add/Edit projects
- Add/Edit experience
- Mobile responsive
- Download CV
- Preview CV
- Toggle CV sections

The UI is based from -> https://app.flowcv.com/resume/content
